# React Reusable Data Table

A reusable `DataTable` component built with React, TypeScript, TanStack Table, Tailwind CSS v4, shadcn/ui, and Lucide icons.

## Features

- Reusable generic `DataTable` component
- Client-side and server-side table support
- Sorting, filtering, pagination, and column visibility controls
- Row selection support
- Loading skeleton state
- Demo showcase page for local development
- `@/*` path alias support

## Project Structure

```text
src/
  components/
    DataTable.tsx
    ui/
  examples/
    ExampleUsage.tsx
  index.ts
```

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:5173/`.

## Build

```bash
npm run build
```

## Notes

- The demo app lives in `src/examples/ExampleUsage.tsx`.
- The reusable export is available from `src/index.ts`.
